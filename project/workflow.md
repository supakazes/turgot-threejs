# Workflow

Colonnes projet :
- Backlog
- Ready
- Todo this week
- In Progress
- In Review
- Blocked
- Done

Règles :
- Un seul ticket en cours.
- Les tickets doivent contenir une section 'Comment faire'.
- Les tickets doivent être réalisables en 1 à 4 heures.
- L'objectif principal reste la démo de septembre.

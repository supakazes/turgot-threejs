# Roadmap Turgot 3D

## Vision

Créer une expérience interactive permettant d'explorer le Paris du XVIIIe siècle à partir du plan Turgot, avec une représentation 3D fidèle et pédagogique.

---

## Phase 1 — Vertical Slice (objectif septembre 2026)

Objectif : démontrer la faisabilité technique et l'intérêt visuel du projet.

Contenu :

- Three.js + TypeScript + Vite
- Pipeline QGIS → Blender → GLB
- Planche 11
- Notre-Dame
- Pont-Neuf
- Palais de Justice
- Place Dauphine
- Seine
- 5 à 8 points d'intérêt
- Interface minimale
- Comparaison avec le Paris actuel

Livrable :

- Démonstration jouable
- Mise en ligne publique
